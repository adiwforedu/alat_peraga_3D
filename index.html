Tujuan Pembelajaran

Melalui media interaktif ini, diharapkan peserta didik Fase E mampu:

  1. Memahami konsep dasar vulkanisme serta karakteristik pergerakan magma dengan benar.
  2. Mengidentifikasi ciri khas dan membedakan 3 bentuk utama gunung berapi (Maar, Perisai, dan Strato).
  3. Menganalisis dampak fenomena erupsi terhadap lingkungan sekitar.
  4. Menerapkan prinsip mitigasi bencana (pra, saat, dan pasca-erupsi) demi keselamatan jiwa.

Petunjuk Interaksi: Gunakan panel simulasi di sebelah kanan untuk mengeksplorasi gunung api secara 3D/AR. Model dapat digeser, diputar, dan diperbesar untuk mengamati morfologinya secara nyata.

Uraian Materi: Gunung Berapi

A. Pengertian Vulkanisme

Vulkanisme merupakan peristiwa naiknya magma dari dalam litosfer menuju ke permukaan bumi melalui celah, patahan, atau pipa kepundan (diatrema). Tenaga dorong ini dipicu oleh akumulasi gas bertekanan tinggi di dalam dapur magma.

B. Bentuk-Bentuk Gunung Berapi

Morfologi gunung api sangat ditentukan oleh kekentalan lava dan kekuatan letusannya:

  • 1. Gunung Api Maar (Kawah): Terbentuk dari sebuah erupsi eksplosif tunggal yang sangat dahsyat, menyisakan lubang kawah besar yang sering kali terisi oleh air (danau kawah). Contoh: Danau Klakah di Lumajang.
  • 2. Gunung Api Perisai (Shield): Terbentuk dari efusi lava yang sangat encer bertekanan rendah, menyebar luas hingga membentuk lereng sangat landai menyerupai tameng. Contoh: Gunung Mauna Loa di Hawaii.
  • 3. Gunung Api Strato (Kerucut): Hasil kombinasi erupsi lelehan (efusif) dan ledakan (eksplosif) yang terjadi secara bergantian secara terus menerus, membentuk struktur berlapis-lapis. Mayoritas gunung api di Indonesia bertipe strato (seperti Merapi, Gede, dan Galunggung).

C. Langkah Mitigasi Bencana Erupsi

Langkah preventif wajib dipahami guna meminimalisir risiko jatuhnya korban:

  • Pra-Erupsi: Mematuhi sistem peringatan dini, mengamati status level aktivitas (Normal, Waspada, Siaga, Awas), serta menyiapkan tas evakuasi darurat.
  • Saat Erupsi: Menghindari kawasan radius bahaya, mengenakan masker/kacamata pelindung debu vulkanik, dan tidak berada di lembah sungai demi menghindari aliran lahar.
  • Pasca-Erupsi: Selalu membersihkan sisa abu di atap rumah agar struktur bangunan tidak roboh akibat beban material.
